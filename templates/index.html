Project Overview

Speech Link is a wireless system that bridges the gap between speech and text by providing real-time speech-to-text and text-to-speech conversion. Powered by an ESP32, it enables remote communication over a wireless network, making interaction easier and more accessible.

System Architecture

  • Browser-based frontend
  • Speech recognition layer
  • Python backend on Raspberry Pi

Objectives

  • Accessibility
  • Low cost
  • Real-time response

Technology Stack

Hardware

  • Microphone
  • Speaker
